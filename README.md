# cub3d

This project uses raycasting to simulate a 3d render with the mlx library

> make bonus && ./cub3d_bonus maps/valid/bonus.cub

### Bonus list:
  - Gates that can be opened and closed
  - Invisibility mode (go through closed gates and boxes)
  - Point of view and weapon selection
  - Sprint
  - Texture transparency (gates)
  - Textures that can be destroyed with adequate weapon
